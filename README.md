# Summary
A Discord bot with basic functions to serve the Silver Fist Dojo server.

# Slash commands
/call - Call Slambot. Mostly used to test the connection.
/show-char-codes-sf6 - Shows the character codes for the characters in Street Fighter 6.
/show-user-codes - Shows the codes for the other server users.
/time-check *user* - Shows the current local time for *user* based on 24 hour clock.
